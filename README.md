# AWS S3 Scanner

A Python-based auditing tool that scans Amazon S3 buckets across all regions to identify empty buckets in your AWS account. Helps cloud teams optimize storage costs, maintain clean infrastructure, and ensure compliance with cloud governance policies.

## Features
- Automated scanning of all S3 buckets across regions
- Identifies buckets with zero objects
- Regional bucket location tracking
- Exports detailed reports for cleanup decisions
- Secure credential management via environment variables
- Efficient API usage with MaxKeys optimization

## Use Cases
- Cost optimization by identifying unused storage resources
- Cloud infrastructure cleanup and maintenance
- Compliance auditing for storage governance
- Security assessments to eliminate unnecessary attack surfaces
- Pre-migration and consolidation planning
